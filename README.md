# Easy-Go: Intelligent Transportation System Prototype

Welcome to the **Easy-Go** Mobile App Prototype! This repository contains a fully interactive, web-based mobile prototype demonstrating the user interface and user experience of a next-generation commute and transportation assistant.

🔗 **Live Demo:** [https://easygoprototype.vercel.app/](https://easygoprototype.vercel.app/)

## 🚀 Features

- **Dynamic Mobile Prototyping Frame**: View the app through a custom-built, interactive phone frame that displays screen metadata, UI flow diagrams, and component details.
- **Smart Commute Dashboard**: Quickly check live routes, weather conditions, and community-reported traffic alerts.
- **Community Reporting**: A built-in feed for users to report and verify road conditions, hazards, and traffic (Heavy Traffic Detection - HTD).
- **Interactive Route Planner**: Simulated turn-by-turn navigation UI with ETA and alternate route options.
- **AI Chat Assistant**: An interactive AI chat interface for users to get personalized commute advice and transit updates.
- **Full Dark Mode Support**: A globally scoped, highly polished Dark Mode that seamlessly transitions the entire UI from a clean light theme to a sleek, dark aesthetic.
- **Notification Center**: Centralized hub for incoming alerts, danger warnings, and passable road verifications.

## 🛠️ Technology Stack

This prototype is built entirely with vanilla web technologies to ensure lightweight performance and maximum flexibility without requiring a build step:

- **HTML5**
- **Vanilla CSS3** (Custom properties, Flexbox/Grid, transitions, and advanced selectors)
- **Vanilla JavaScript** (ES6+, DOM manipulation, Template Literals)
- **FontAwesome 6** (Icons)
- **Google Fonts** (Inter and Outfit)
